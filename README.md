## 컴파일 및 실행

### 컴파일
```bash
clang++ -g 2025_2_CSE103_00.cpp -o 2025_2_CSE103_00
```

### 실행
```bash
./2025_2_CSE103_00
```

## 파일 구조
- `2025_2_CSE103_00.cpp`: 메인 소스 코드
- `.gitignore`: 불필요한 파일 제외 설정

- 배열 인덱스는 0부터 시작
- 문자열은 null terminator(`'\0'`)로 끝남
- 참조 매개변수(`&`)는 원본 변수 수정
